# Week 1 — Scaffold

**Tag:** `v0.0.1-scaffold` (proposed)
**Status:** ✅ green

## Goal

Repo builds. CI green. `pnpm tauri dev` opens a window calling `invoke("ping")`. `curl localhost:5167/health` returns `{"ok": true}` when sidecar is started manually.

## Demo recipe

```bash
# 1. Bootstrap (idempotent)
./scripts/bootstrap.sh

# 2. Start sidecar manually
cd backend && uv run python -m meetwit
# → http://127.0.0.1:5167
curl -s http://127.0.0.1:5167/health
# → {"ok":true,"version":"0.0.1"}

# 3. Start the desktop app
pnpm tauri:dev
# → window opens with "Meetwit" + "Ping Rust core" button
# → click button → "✓ pong from Meetwit Rust core"
```

## Open issues / deferred to next weeks

- **Sidecar auto-spawn** — currently manual. → Week 2.
- **Database schema** — `db.py` has only the engine factory. → Week 3.
- **Icons are placeholders** — solid-green squares generated programmatically. Replace with a real brand mark before W14.
- **`cargo deny` not installed locally** — CI runs it via the EmbarkStudios action. Optional locally: `cargo install cargo-deny`.
- **No actual unit tests yet** — sidecar smoke test (2 tests) is the floor. Add more as features land.
- **Tauri `frontendDist` requires a prior Vite build** — by design, but means first-time contributors must run `pnpm build` once before `cargo check`. Captured in BUILDING.md.

## What we proved

1. The three toolchains coexist on macOS arm64 without conflict.
2. Tauri 2 + Vite + Tailwind 4 + TanStack Router pipeline works end-to-end.
3. FastAPI + sqlite-vec wheel installs from PyPI on Apple Silicon without source builds.
4. `mypy --strict` is achievable from day 1 (key dependencies have stubs or `ignore_missing_imports`).
5. The Tauri ↔ React `invoke()` round-trip works in dev (validated manually with `pnpm tauri:dev`).

## What we did NOT prove (yet)

- The full release `tauri build` pipeline (codesigned .app + .dmg) — that's Weeks 13-15.
- Auto-spawn lifecycle (Tauri Rust → Python sidecar) — Week 2.
- That whisper-rs compiles with Metal + CoreML features on this host — Week 7 spike.
