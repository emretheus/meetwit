# Week 14 — Onboarding + permissions + release-mode sidecar

**Status:** ✅ green

## Goal

7-step onboarding screen, Whisper model auto-download with progress, permission deep-links, and release-mode `SpawnOptions` that uses the bundled PyInstaller binary.

## What landed

**Backend (Rust):**
- `SpawnOptions::release(resources_dir)` — points at `Contents/Resources/python-backend/meetwit-sidecar` (the path the W13 `build-release.sh` deposits the bundle at)
- `build_spawn_options()` in `lib.rs` picks dev vs release automatically: looks for the bundled binary next to the current exe; falls back to `uv run python -m meetwit` if missing
- Two new IPC commands:
  - `whisper_download(model)` — streams a HuggingFace ggml model, emits `whisper-download-progress` events with `{bytes_done, bytes_total, finished}`
  - `open_system_settings(pane)` — opens the macOS System Settings deep-link for `microphone`, `screen-recording`, or `privacy`
- Deps added: `tokio-stream`, `futures-util`; `reqwest` extended with `stream` feature

**Frontend:**
- `desktop/src/routes/onboarding.tsx` — `/onboarding` route, 6-step wizard:
  1. Welcome (privacy promise)
  2. Microphone (tries `mic_start` to trigger TCC prompt)
  3. Screen Recording (deep-link + clear "fully quit and relaunch" warning)
  4. Ollama check (`/memory/llm/status`)
  5. Whisper model download (`tiny.en`, ~75 MB, with live progress bar)
  6. Ready (status summary + go-home button)
- Progress bar across the top tracks the step.

## Open follow-ups

- No auto-redirect from `/` to `/onboarding` for first-time users — V1.1 adds a `~/Library/Application Support/Meetwit/.onboarded` marker file check.
- Permission state isn't queried (the app just tries to start the mic and surfaces failure). macOS doesn't have a clean "is mic granted?" API outside AVFoundation; pragmatic for V1.
- "Download model" runs once on the Tauri side; if the user closes the app mid-download, partial file is left. Acceptable: deleting the half-file from `~/Library/Application Support/Meetwit/models/` retries cleanly.
